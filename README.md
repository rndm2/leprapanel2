# LepraPanel2 — это старая добрая навигационная панель для Лепрозория по-новому

Мы взяли старую навигационную панель, переписали её, добавили несколько юзерскриптов и получилась LepraPanel2. Она опять умеет проверять карму, рейтинг, посты и инбоксы через заданный промежуток времени.
На данный момент поддерживается только Mozilla Firefox. Поддержка Google Chrome была в планах но сейчас стала не актуальна из-за альтернативной панели для Chrome от других чуваков, а также того факта, что в Chrome сделать полноценную панель не представляется возможным (только кнопки).
